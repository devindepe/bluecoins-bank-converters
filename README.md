# bluecoins-spanish-bank-converters
Conjunto de scripts en Python para convertir extractos bancarios de bancos españoles (Ibercaja, BBVA, etc.) al formato CSV estándar compatible con la App Bluecoins. Facilita la importación de movimientos manteniendo la integridad de fechas, importes y saldos.  ¡Contribuciones para nuevos bancos son bienvenidas!
